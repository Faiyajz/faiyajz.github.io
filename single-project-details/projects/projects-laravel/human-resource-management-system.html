Human Resource Management System

Maven Design Limited

Description

A Complete Human Resource Management Solution, can be integrated with a device to get attendance through finger print.

The project was built with PHP, Laravel, Javascript, MySQL, JSON for MAVEN DESIGN LIMITED.

Main Features

Employee Management, User Management, Generate report and download the report, Payroll Management, Notification, Attendance Management, Device Management, Audit Management, File Management etc. are the main features of this web application.
